# Utils

This directory contains utility classes and helper functions.

## Structure

Utilities provide common functionality used throughout the application.

### Files:
- `constants.dart` - App-wide constants (✓ Created)
- `theme.dart` - Theme configuration (✓ Created)

### Planned Utilities:
- `validators.dart` - Input validation functions
- `formatters.dart` - Data formatting helpers (dates, currency, etc.)
- `image_helper.dart` - Image compression and processing
- `navigation_helper.dart` - Navigation utilities
- `logger.dart` - Logging utilities

Each utility should:
- Provide static methods or functions
- Be stateless and pure where possible
- Be well-tested
- Have clear documentation
